The following new files are available on the eGRID web site (epa.gov/energy/egrid):

These files represent eGRID2016.

  egrid2016_all_files.zip contains all the files released for eGRID2016 including operational data for year 2016:

			egrid2016_data.xls
			egrid2016_data_Metric.xls
			egrid2016_summarytables.pdf
			egrid2016_summarytables.xls
                     	egrid2016_egrid_subregions.jpg
			egrid2016_nerc_regions.jpg
			egrid2016_technicalsupportdocument.pdf
			egrid2016_sequencenumbers.xlsx
			egrid_release_notes_2018-02-15.txt (this file)


  eGRID_All_Files_1996_thru_2016_data.zip contains all files from all editions of eGRID including operational data for the following years:
  1996, 1997, 1998, 1999, 2000, 2004, 2005, 2007, 2009, 2010, 2012, 2014, and 2016.
