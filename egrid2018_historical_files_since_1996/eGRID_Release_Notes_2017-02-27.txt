The following new files are available on the eGRID web site (epa.gov/energy/egrid):

These files represent version 2 of eGRID2014.

  eGRID2014_data_v2.zip contains all version 2 files released for eGRID2014 including operational data for year 2014:

			eGRID2014_Data_v2.xls
			eGRID2014_Data_Metric_v2.xls
			eGRID2014_GHGOutputRates_v2.pdf
			eGRID2014_SummaryTables_v2.pdf
			eGRID2014_SummaryTables_v2.xls
                     	eGRID2014_eGRID_subregions.jpg (unchanged from version 1)
			eGRID2014_NERC_regions.jpg (unchanged from version 1)
			eGRID2014_TechnicalSupportDocument_v2.pdf
			eGRID_Release_Notes_2017-02-27.txt (this file)


  The version 2 files replace the following original eGRID2014 files, which were released 1/13/2017:

			eGRID2014_Data.xls
			eGRID2014_Data_Metric.xls
			eGRID2014_GHGOutputRates.pdf
			eGRID2014_SummaryTables.pdf
			eGRID2014_SummaryTables.xls
                     	eGRID2014_eGRID_subregions.jpg
			eGRID2014_NERC_regions.jpg
			eGRID2014_TechnicalSupportDocument.pdf
			eGRID_Release_Notes_2017-01-13.txt


  eGRID_All_Files_1996_thru_2014_data.zip contains all files from all editions of eGRID including operational data for the following years:
  1996, 1997, 1998, 1999, 2000, 2004, 2005, 2007, 2009, 2010, 2012, and 2014.
